# PRD: 刷题工具

## 概述

面向大学生的刷题工具，支持导入题库、自动识别题目（含图片）、记录错题、收藏题目。作为工具集的一部分，通过统一认证系统登录。PWA 可在手机安装。

## 数据库

与资讯项目共用 Supabase 实例（ftulngtxspkkmxjanegw.supabase.co），共用 `users` 表。新增 `quiz_*` 表。

## 核心功能

### P0 — 基础功能

- **登录认证**：调资讯项目 API `/api/auth/login`，JWT token 本地存储
- **题库列表**：展示默认题库 + 用户自建题库，可搜索筛选
- **文件导入题库**：上传 Excel/CSV/Word/PDF → 解析 → 预览 → 确认入库
- **题目总览**：虚拟滚动列表，可跳题号，显示题目摘要
- **顺序刷题**：按题库顺序，记忆进度，下次继续
- **随机刷题**：打乱顺序，可限定题数
- **错题本**：查看某 Deck 的错题，手动移除，显示错误次数
- **错题重刷**：只刷某 Deck 的错题，连续做对不自动移除
- **收藏**：标记题目，独立于错题
- **收藏重刷**：只刷某 Deck 的收藏题
- **图片题支持**：题目和选项可含图片，ZIP 打包导入
- **题库编辑**：低调入口，修改答案/解析

### P1 — 进阶功能

- **模拟考试模式**：限时、交卷后出分、不显示逐题答案
- **搜索题目**：跨题库搜索
- **刷题统计**：Session 历史 + 正确率趋势
- **分享题库**：导出为文件，供他人导入

### P2 — 未来扩展

- 填空题模糊匹配（关键词匹配）
- AI OCR 识别扫描 PDF/拍照
- 社区题库共享
