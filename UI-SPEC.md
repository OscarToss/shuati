# UI-SPEC: 刷题工具

## 设计语言

| 维度   | 选择                               | 理由                                         |
| ------ | ---------------------------------- | -------------------------------------------- |
| 风格   | Flat Design Mobile (Touch-First)   | 移动端 PWA，无阴影零高程，纯色分区，性能最优 |
| 字体   | Inter                              | 中性专业，可读性高，适配长文本题目           |
| 主色   | #2563EB (蓝)                       | 冷静、信任、学习场景                         |
| 正确色 | #10B981 (绿)                       | 答题正确反馈                                 |
| 错误色 | #DC2626 (红)                       | 答题错误反馈                                 |
| 强调色 | #F59E0B (金)                       | 排行榜/收藏高亮                              |
| 圆角   | 6px(小) / 12px(卡片) / 999px(按钮) | 微圆不尖锐，不卡通                           |
| 间距   | 4/8/16/24/32/48 体系               | 标准 8px 递增                                |
| 触控   | ≥48px 所有可点元素                 | 移动端规范                                   |
| 图标   | Lucide Icons                       | SVG 矢量，统一 stroke                        |

## 通用规则

- 无阴影 (shadow: none / elevation: 0)
- 纯色分区替代分割线
- 按钮按压反馈: scale 0.97 + 颜色变深 (150ms)
- 页面过渡: 150-250ms ease
- 骨架屏加载，不空转圈
- 禁止 emoji 做图标

## 页面布局

### 题库列表页

- 顶部搜索栏 (sticky)
- 默认题库 + 我的题库分区，纯色背景交替
- 题库卡片: 标题 + 题数 + 进度条
- 底部导航栏: 题库 / 统计 / 我的

### 刷题页 (核心)

- 顶部: 进度条 + 题号 + 操作溢出菜单(收藏/举报)
- 中部: 题目卡片 (题干 + 图片 + 选项)
- 底部: 选项按钮区 (单选大色块 / 判断对错 / 填空输入)
- 答后: 卡片变色 (绿对/红错) + 解析展开 + 下一题
- 滑动手势: 左右切题

### 题目总览页

- 虚拟滚动列表
- 每行: 题号 + 题型图标 + 题干截断 + 状态(正确/错误/未做)
- 底部: 浮动"开始刷题"按钮

### 错题本/收藏页

- 筛选工具栏 (题型/标签)
- 题目卡片列表，左滑删除(错题)
- 空态引导文案

### 导入页

- 上传区 (拖拽/点击)
- 预览表 (绿色正常/黄色重复/红色异常)
- 底部: 确认入库按钮 + 下载修正模板

## 关键交互

### 答题反馈

```
用户选择 → 选项变蓝(选中态) → 150ms → 蓝变绿(正确)/蓝变红(错误)
→ 解析区域滑入展开 → 300ms 后底部出现"下一题"按钮
```

### 文件导入

```
拖拽/选择文件 → 进度条动画 → 预览表(虚拟滚动) → 逐行标注状态
→ 用户修或跳过 → 确认 → toast"导入成功，共 N 题"
```

### 离线提示

- PWA 已缓存: 不显示
- 联网但弱网: 顶部轻提示"网络较慢"
- 断网 + 非缓存页: 全屏提示"离线中，查看已下载题库"
