# CONTEXT: 刷题工具

## 术语表

| 中文     | 英文         | 定义                                                                                                           |
| -------- | ------------ | -------------------------------------------------------------------------------------------------------------- |
| 题库     | Deck         | 题目的集合，如"四级词汇"、"考研政治马原"。分默认题库（系统提供，所有人可读）和自建题库（用户导入，仅本人操作） |
| 题目     | Question     | 单道题，含题干、选项、答案、解析、标签、图片                                                                   |
| 题型     | QuestionType | choice（单选）、multi（多选）、truefalse（判断）、fill（填空）                                                 |
| 刷题会话 | Session      | 用户在某 Deck 中的一轮做题过程，记录对/错/耗时                                                                 |
| 错题     | Wrong        | 用户答错的题目，手动管理。记录累计错误次数                                                                     |
| 收藏     | Bookmark     | 用户主动标记的题目，与错题独立                                                                                 |
| 进度     | Progress     | 用户在某 Deck 的做题进度（做对/做错总数）                                                                      |
| 默认题库 | Default Deck | 系统内置，user_id = NULL，is_default = true，RLS 所有人可读                                                    |
| 图片引用 | Image Ref    | 题目中 `[[img:xxx.png]]` 格式引用图片文件，与题库文件同 ZIP 导入                                               |
